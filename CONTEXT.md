# study-eng-h

한국어 사용자 대상 영어 어휘 학습 서비스. CEFR 진단으로 레벨을 정하고, 데일리 퀴즈와 플래시카드(SRS)로 어휘를 암기·유지시키는 단일 컨텍스트다.

## Language

### 학습 루프

**오답 편입 (Enrollment)**:
퀴즈에서 틀린 단어가 자동으로 복습 큐에 등록되는 것. 오답만 편입되며, 정답·전체 힌트 정답은 편입되지 않는다.
_Avoid_: 동기화, 연동, 오답 노트

**복습 큐 (Review Queue)**:
복습 시점이 도래해 사용자가 오늘 복습해야 하는 단어의 집합. 플래시카드 복습 세션이 이 큐를 소비한다.
_Avoid_: 복습 리스트, due 목록

**신규 카드 풀 (New Card Pool)**:
아직 학습을 시작하지 않은 단어의 집합. 한 번이라도 학습 이력이 생긴 단어(오답 편입 포함)는 이 풀에서 빠진다.
_Avoid_: 새 단어 목록

**추가 연습 (Extra Practice)**:
하루 첫 퀴즈 완료 이후의 반복 제출. 보상(XP·보너스)은 없지만 학습 신호는 유효해 오답 편입은 일어난다.
_Avoid_: 재도전, 복습 퀴즈

**미연결 단어 (Unlinked Word)**:
퀴즈에만 존재하고 어휘 사전에 없는 단어(관용구 등, 퀴즈 단어의 약 20%). 오답이어도 편입되지 않는다.
_Avoid_: 미등록 단어

## Flagged ambiguities

- **"복습"**: 플래시카드의 복습 세션(모드)과 "복습 시점 도래"라는 상태 개념이 같은 단어로 불린다.
  세션을 가리킬 때는 "복습 세션", 상태를 가리킬 때는 "복습 도래"로 구분한다.

## Example dialogue

> **Dev**: 퀴즈에서 틀린 단어는 어떻게 되나요?
> **Domain expert**: 오답 편입이 일어나요. 어휘 사전에 있는 단어면 복습 큐에 들어가고, 다음날부터 복습 도래 상태가 됩니다. 미연결 단어는 편입되지 않아요.
> **Dev**: 추가 연습에서 틀려도요?
> **Domain expert**: 네. 추가 연습은 XP만 없을 뿐 학습 신호로는 동일해서 편입은 똑같이 일어납니다.
> **Dev**: 편입된 단어를 플래시카드 신규 세션에서도 만나나요?
> **Domain expert**: 아니요. 편입되는 순간 학습 이력이 생기니 신규 카드 풀에서는 빠지고, 복습 세션에서만 만납니다.
